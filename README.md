🎉 Positive Congratulation

Landing page simples desenvolvida com HTML e CSS com foco em design moderno e responsividade.

📌 Sobre o projeto

O Positive Congratulation é uma landing page criada para praticar conceitos de front-end, com foco em estrutura semântica, estilização e adaptação para diferentes dispositivos.

🚀 Tecnologias utilizadas

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![Responsive Design](https://img.shields.io/badge/Responsive-Design-00C853?style=for-the-badge)
![Status](https://img.shields.io/badge/status-finished-brightgreen?style=for-the-badge)

O projeto simula uma página de mensagem positiva/congratulativa, com um layout simples, porém visualmente agradável.
💻 Funcionalidades
- Interface limpa e intuitiva
- Estrutura organizada em HTML
- Estilização com CSS
- Layout responsivo para diferentes telas

📱 Responsividade

A aplicação foi desenvolvida pensando em usabilidade tanto em dispositivos móveis quanto em desktops, garantindo uma boa experiência em qualquer tela.
